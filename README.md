# MVP - Análise exploratória de dados e boas práticas

Dados sobre resultados de estudantes em escolas secundárias de duas escolas de Portugal (Gabriel Pereira ou Mousinho da Silveira) nas avaliações do 1º (G1), 2º (G2) e 3º (G3) períodos do último ano do ensino médio nas matérias de matemática e português. O dataset possui 30 variáveis relacionadas a questões demográficas, sociais e escolares dos alunos e foram obtidas mediante relatórios e questionários escolares, tendo sido publicadas em https://archive.ics.uci.edu/dataset/320/student+performance.

Este projeto tem como objetivo realizar uma análise exploratória dos dados com avaliação das variáveis mais importantes para o desempenho escolar dos estudantes da amostra obtida.
